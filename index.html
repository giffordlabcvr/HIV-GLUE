HIV-GLUE

A GLUE project for the human immunodeficiency viruses.

Download .zip Download .tar.gz View on GitHub

Description

This is HIV-GLUE: a GLUE project for the human immunodeficiency viruses (HIVs).

The human immunodeficiency viruses HIV-1 and HIV-2 cause acquired immunodeficiency syndrome (AIDS).

GLUE is an open source, data-centric bioinformatics environment specialised for developing virus genome data resources (VGDR).

This GLUE project contains reference information for HIV-1 and HIV-2, including:

  1. A set of HIV-1 reference sequences (i.e. existing HIV-1 sequences that are well-annotated) with associated metadata.
  2. A list of genome features and their specific locations on full genome reference sequences.

Who can use this resource, and for what?

HIV-GLUE is designed to provide a foundation for the development of a diverse range of genomic resources for HIV-1 and HIV-2.

GLUE projects can extend HIV-GLUE with new data and functionality. So far we have used HIV-GLUE to develop the following GLUE extension projects.

  • HIV-PAIRS - a GLUE project for HIV transmission pair data (Private Repo).
  • HIV-CRNA - a GLUE project in collaboration with the Centre for HIV RNA studies (Private Repo).

Installation

You can install HIV-GLUE on computers running Windows, MacOSX or Linux.

1. Install GLUE, based on the GLUE installation instructions.
2. Once GLUE is installed and working, clone the HIV-GLUE repository into your `gluetools/projects` directory.
3. Within the `gluetools/projects/HIV-GLUE` directory, start GLUE and build the HIV-GLUE extension by issuing the following command in GLUE: 

Mode path: /
GLUE> run file hiv.glue
